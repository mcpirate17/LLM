### CKA & Novelty
- CKA references: `artifacts/cka_references/v1/`. Runtime auto-loads via `ReferenceCkaStore`. Regenerate with `python tools/export_cka_references.py`.
- Novelty: `NoveltyMetrics.novelty_confidence` (0.0–0.9). Grammar weights use `1.0 + novelty * confidence`.
- Breakthrough promotion: thresholds match `Aria.PUBLICATION_*` (baseline < 0.90, std ≤ 0.03, ≥ 5 seeds, OOD ≥ 0.67, HP ≥ 0.75, novelty_confidence ≥ 0.5).

### HYDRA Integration
- `RunConfig.data_mode="hydra"` uses HYDRA's `create_universal_loader`. Token IDs projected via `% vocab_size` when HYDRA vocab (GPT-2 50257) exceeds model vocab.

### Scope
- The AI scientist **only researches neural network layer architectures** — it generates random compositions of ops, tests if they compile and learn, and evolves the grammar toward successful patterns.

### Control & Observability
- Every 5th continuous run uses default grammar weights (`control_experiment=true`).
- Synthesis results include `generated_op_distribution` and `generation_distribution_shift` for grammar feedback verification.
