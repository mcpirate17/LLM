## Notes

### Technical Conventions
- `scientist/analytics.py` imports `get_primitive` and `GrammarConfig` at module level
- `synthesis/graph.py` ComputationGraph has `_cache` dict cleared by mutation -- use `add_op`/`set_output`
- `database.py` ExperimentDB has `batch()` context manager for deferred commits
- Tests: `/home/tim/venvs/llm/bin/pytest tests/ -x` (220 passing as of 2026-02-16)
- CLI: `python -m research --mode=synthesize` starts via `start_experiment/start_continuous/start_evolution`

### CKA & Novelty
- CKA references: `artifacts/cka_references/v1/`. Runtime auto-loads via `ReferenceCkaStore`.
- Novelty: `NoveltyMetrics.novelty_confidence` (0.0–0.9). Grammar weights use `1.0 + novelty * confidence`.
- Breakthrough: thresholds match `Aria.PUBLICATION_*` (baseline < 0.90, std ≤ 0.03, ≥ 5 seeds, OOD ≥ 0.67, HP ≥ 0.75, novelty_confidence ≥ 0.5).

### HYDRA Integration
- `RunConfig.data_mode="hydra"` uses HYDRA's `create_universal_loader`. Token IDs projected via `% vocab_size`.

### Scope
- The AI scientist **only researches neural network layer architectures** — generates random compositions of ops, tests if they compile and learn, evolves grammar toward successful patterns.

### Control & Observability
- Every 5th continuous run uses default grammar weights (`control_experiment=true`).
- Synthesis results include `generated_op_distribution` and `generation_distribution_shift`.
