# Archive Scripts

This directory is a quarantine area for historical maintenance scripts. Files here are not part of the active `research` runtime contract.

Rules:

- Do not import these scripts from production or test code.
- Do not assume the scripts are current, supported, or covered by CI.
- If a script must become active again, move it into an owned runtime/tooling location and add tests in the same change.
- Prefer deleting archived scripts that no longer have an owner or a documented operational use.
